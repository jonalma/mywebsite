After being busy with school and work, I have found some time and motivation to create my personal website. I've been wanting to code up my own website for some time because I've been fascinated with Javascript and its frameworks. As a kid, I would go on Xanga (an old blog which was equivalent to Facebook at the time) and make my own front-end layouts with cool music, backgrounds, and text.

This project gave me an opportunity to get down with JavaScript, jQuery, CSS, and HTML. I aimed to make my website interactive, creative, simplistic, and boring (just kidding). The photos below are arranged using a jQuery plug-in, CollagePlus by Ed Lea. I handled the events (Popups, transitions, etc.) with a mix of Javascript and jQuery (and jQuery Mobile framework). Thanks to the jQuery/JavaScript documentation and Stack Overflow for the help.

My name is Jon Almazan, and I recently graduated with a BS in Computer Science at California State University, Long Beach. Some areas that interest me are computer networking, systems adminstration, and web applications.

I enjoy basketball, rock climbing, quality time family and friends, In-N-Out, all-you-cant-eat buffets, music, and anything that is fun, funny, or thrilling. If you have not noticed, I have placed some of my favorite images that I appreciate.
